## Projects

### Network Traffic Analysis

Analysing packet captures using Wireshark to identify suspicious activity, anomalies, and potential indicators of compromise.

### Vulnerability Assessment

Using tools such as Nmap and Nessus to identify open ports, system weaknesses, and recommend remediation strategies.

### Digital Forensics Investigation

Examining system logs and artefacts to detect and analyse potential security incidents.
